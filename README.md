---
此笔记只是本人学习过程中的一点知识归集，并不一定会列出所有语言的所有知识点，也不会对语言进行任何介绍，目前是学懂一点，记录一点，欢迎各位童鞋一起来记录，也欢迎转给更多的人学习。
